A mini e‑commerce website where users can browse products, view details, add items to a cart, and place demo orders. Built with HTML, CSS, JavaScript and React as a simple online store example.

# Mini E‑Commerce Website

A simple mini e‑commerce website built to practice front‑end and basic e‑commerce logic. It showcases my ability to build responsive UI, work with product data, and implement a shopping cart.

## What I built

- Product listing page with prices and images.
- Product detail view.
- Shopping cart with add/remove/update quantity.
- Basic checkout / order summary (demo only)
- No payement intergration

## Tech skills demonstrated

- HTML & CSS for layout and responsive design.
- JavaScript for dynamic UI and cart logic.
- React for reusable component 
  

## Why this project

This project shows how I structure a small web app, organize components and files, and handle state (cart, products) in a clear and maintainable way.
